Using AFIO really slows down my compile times. Can't you do something about that?

We know about that slowdown! The solution is easy: define BOOST_AFIO_HEADERS_ONLY to 0 for each compiland, and link to the AFIO shared library instead. Do the same for ASIO by defining BOOST_ASIO_HEADER_ONLY to 0 for each compiland, and link to the ASIO shared library instead.

You should notice a 6x-10x improvement to compilation time, which is a huge win during debugging.
