truncate (single)

Schedule an asynchronous data write after a preceding operation.

Synopsis 
async_io_op async_file_io_dispatcher_base::truncate(const async_io_op & op, off_t newsize)
Parameters

Type

Concept

Name

Description

const async_io_op &

op

An op handle.

off_t

newsize

The new size for the file.

Returns

An op handle.

Header

#include <boost/afio/afio.hpp>

Complexity

Amortised O(1) to dispatch. Amortised O(1) to complete if truncating file lengths is constant time.

Exception Model

Propagates exceptions of any input preconditions with an errored state at the point of dispatch, and throws a std::runtime_error if any inputs have values which could not possibly be correct.
